## Install Go

[Instructions](https://go.dev/dl/) on installing Go 1.20.x

## Install Go packages

```bash
go mod vendor
```

## Run tests

```bash
go test .
```

## Run tests with log

```bash
go test . -v
```

## Refresh go.mod

```bash
go mod tidy
```
